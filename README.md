# Travel Website

A static travel website built using HTML and CSS.  
This project demonstrates web design skills and photo graphy skill, clean layout for travel-related content.

 Tech Stack
- HTML
- CSS

 Features
- Image gallery
- Navigation menu
- responsive web design
- Attractive UI design
- Multiple sections 

Project Purpose
This project is created to showcase front-end design and layout skills.  
It is suitable for portfolio display and demonstrates basic web development abilities.
